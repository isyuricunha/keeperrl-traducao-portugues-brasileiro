# KeeperRL - Tradução para Português Brasileiro

**Se essa tradução te ajudou, apoie via Pix:** `pix@yuricunha.com`

Tradução do jogo **KeeperRL** para **Português Brasileiro (pt-BR)**.

- **Status:** completa e revisada

## Instalação

1. Localize a pasta de instalação do KeeperRL.
1. Copie o arquivo `portugues.txt` para:

```text
KeeperRL\data_free\game_config\translations
```

1. Inicie o jogo e selecione o idioma (se aplicável) ou mantenha a tradução instalada na pasta de traduções.

## Download

- Baixe o `portugues.txt` na aba **Releases** do GitHub (seção **Assets/Downloads**).
